## Honest state of this build

These docs describe the **June 30 Stellar Level 3 + APAC submission scope**: testnet only,
Freighter only, Philippines only, anchor **simulated**. The single riskiest assumption — that a
regulated institution will accept the credential as preferred intake (idea.md §9) — is **not**
tested by this build. Material open questions (contract authorization model, issuer-key custody,
off-chain document storage + access control, RA 10173 retention/erasure vs on-chain permanence, and
the missing test/CI tooling) are listed in each doc's "Open questions" section, not papered over.
